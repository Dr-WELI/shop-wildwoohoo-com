# Product page template

This folder holds the template for a single product page. Clone `product-page.html` when you add a new product.

## Steps to add a new product

1. **Choose the category** the product belongs to: `childrens-books`, `educational`, `merch`, or `tools`.
2. **Create a folder** at `/categories/[category]/[product-slug]/` and copy `product-page.html` into it as `index.html`.
3. **Search for `REPLACE-ME`** in the new file and fill in every placeholder. Search for `{{` too if any remain.
4. **Drop product images** into `/assets/img/[product-slug]/`. Seven images, named `01-hero.jpg` through `07-model.jpg`. The order matters; see Insight 6 in the shop immersion notes for the rationale.
5. **Add the new URL** to `/sitemap.xml` with today's `<lastmod>` date.
6. **Add a product card** on the relevant `/categories/[category]/index.html` page so the product is discoverable from the category listing.
7. **Editorial chain.** Full chain (writer -> reviewer -> chief-editor) before the page goes live. See `~/Studio/CLAUDE.md`.

## Data fields the template expects

| Field | Type | Example | Notes |
|---|---|---|---|
| `name` | text | `Kangaroo Time Tee` | The product name as the customer searches |
| `variant` | text | `Forest Green / M` | Optional variant label |
| `sku` | text | `KT-TEE-FG-M` | Internal SKU, also used in JSON-LD |
| `price` | AUD | `45` | Round AU$ figure per Insight 8 |
| `price-compare-at` | AUD | optional | Only for documented discounts per Insight 21 |
| `description` | text | 80-300 words | Length depends on product type per Insight 34 |
| `images` | 7 paths | `/assets/img/[slug]/01-hero.jpg` ... `07-model.jpg` | Per Insight 6 |
| `material` | text | `Stanley/Stella STTU755 Creator 2.0, 220gsm organic combed cotton, GOTS certified` | Spec + supplier per Insight 2 |
| `fit` | text | `Regular fit. Model is 178cm in size M.` | Or dimensions for non-apparel |
| `made-in` | text | `Printed in Brisbane, Australia by Printful AU` | Country + facility |
| `ships-from` | text | `Sydney, Australia` | City + country |
| `care` | text | `Machine wash cold, hang dry, iron inside out at low heat. Hang-drying saves ~50% of the garment's lifetime carbon footprint per the Hot or Cool Institute 2024 study.` | Specific + sustainability rationale per Insight 25 |
| `fulfillment` | enum | `pod` / `inventory` / `digital` | Branches the copy register per Insight 22 |
| `weight-grams` | number | `220` | For shipping calculations |
| `dimensions-mm` | text | `420 x 297 x 2` | Length x width x height |
| `eligible-regions` | list | `AU, NZ, EU, US` | Constrained by EU GPSR for inventory products |
| `age-recommended` | text | `6+` | Only for kids products per Insight 23 |

## Title convention (per Insight 4)

`WildWooHoo | [Product Name] | [Variant]`

Max 70 chars for the Google product card. Aim 55-65 incl separators.

## Meta description (per Insight 5)

150-160 chars exactly. Hand-written. Never auto-generated by Shopify or any other tool. Pattern: `[product type]` + `[one specificity competitor lacks]` + `[soft commitment: price, ships-from, or use-context]`.

## Copy length (per Insight 34)

- Tees / posters / stickers / totes: 80-120 words body copy
- Zines / CDs / vinyl / books: 200-300 words body copy

## Pricing (per Insight 8)

Round AU$ figures. AU$45 not AU$44.99. Never hard-code price in body copy (per Insight 27).

## Sustainability claims (per Insights 2, 3, 7)

Name the supplier / spec / certification inline. No vague claims. Acknowledge at least one limit inline if positioning as eco/sustainable.

## What this template does NOT do

- It does not include a real cart, checkout, payment, or fulfillment path. Those arrive when the shop migrates to its e-commerce backend (Shopify per the strategy doc, decision still open per WELI).
- It does not auto-render the price across currencies. That arrives with Shopify Markets.
- It does not load any analytics pixels. GA4, Meta Pixel, TikTok Pixel and Plausible get added at backend-migration time, not now.
